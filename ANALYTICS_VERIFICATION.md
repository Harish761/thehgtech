## üö® TROUBLESHOOTING

### Not seeing data in Realtime?
1. **Wait 30 seconds** - Data isn't instant
2. **Refresh the report** - Click refresh icon
3. **Check you're on the right property** - Look for "G-XL6RCXZJE2"
4. **Disable ad blocker** - uBlock, AdBlock can block Analytics
5. **Try incognito mode** - Browser extensions might interfere

### Seeing duplicate events?
1. **Check you only have ONE gtag.js script** - View page source
2. **Clear browser cache** - Hard refresh (Ctrl+Shift+R)
3. **Check for other tracking codes** - Only one GA4 property

### Data looks wrong?
1. **Filter out your own traffic:**
   - Admin ‚Üí Data Streams ‚Üí Configure tag settings
   - Show advanced settings ‚Üí Define internal traffic
   - Add your IP address
2. **Wait 24-48 hours** - Initial data can be inconsistent
3. **Check timezone settings** - Should be India (GMT+05:30)
