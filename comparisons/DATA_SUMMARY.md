### Deployment Options
**Splunk:**
- On-prem: Physical/virtual appliances
- Cloud: Splunk Cloud (fully managed SaaS)
- Hybrid: Supported

**Elastic Security:**
- On-prem: Self-hosted (open source)
- Cloud: Elastic Cloud (managed service)
- Hybrid: Supported

**LogRhythm:**
- On-prem: Physical appliances or software
- Cloud: LogRhythm Axon (SaaS)
- Self-managed cloud: Any cloud provider
- Hybrid: Supported

### Pricing Models
**Splunk:**
- Data ingestion-based pricing
- Can be expensive at scale
- Starting: Enterprise pricing (contact sales)

**Elastic Security:**
- Open-source core (free)
- Enterprise features require license
- More cost-effective for large data volumes
- Cloud: Subscription-based

**LogRhythm:**
- "True Unlimited Data Platform" (no data volume tiers)
- Starting: ~$2,000/year (varies by size)
- Subscription or perpetual license options
- Cloud add-ons: ~$3,890/year for 1TB storage
