Deepfake Defense: Technical Detection of Synthetic Identities

Forensic techniques for detecting AI-generated audio, video, and images in enterprise environments

25 min read | Intermediate to Advanced | January 2026
Threat Context: AI-driven phishing attacks using deepfakes have increased 1,265% since 2023. In 2025, 73% of organizations reported being affected by cyber-enabled fraud involving synthetic media (WEF 2026 Global Cybersecurity Outlook).
Abstract: This guide provides technical methodologies for detecting synthetic media—deepfake videos, cloned voices, and AI-generated images—in enterprise security contexts. We move beyond policy recommendations to hands-on forensic analysis techniques using open-source tools. Content covers: audio spectral analysis for voice clone detection, facial landmark inconsistency analysis, metadata forensics, and integration into SOC workflows. Target audience: SOC analysts handling fraud investigations, threat hunters analyzing BEC campaigns, and security managers evaluating identity verification systems.

2. Enterprise Attack Scenarios

Scenario 1: CEO Fraud via Voice Clone

Attacker clones the CEO's voice from earnings call recordings (publicly available). Calls the CFO claiming to be traveling, requests urgent wire transfer to "close an acquisition under NDA." The voice is indistinguishable from the real CEO.

Scenario 2: Video Conference Impersonation

Attacker joins a Zoom call using real-time deepfake software, impersonating an executive. Other participants see a convincing video of the "executive" authorizing sensitive actions.

Scenario 3: Synthetic Employee Onboarding

Threat actor creates AI-generated profile photos and forged documents to pass remote verification. Gets hired with access to internal systems—an insider threat from day one.

3. Audio Deepfake Detection

Voice clones leave detectable artifacts. The following techniques can identify synthetic audio:

Spectral Analysis Indicators

Indicator Real Audio Synthetic Audio
High-Frequency Content Natural variation above 8kHz Often missing or artificial above 8kHz
Breath Sounds Natural inhales between phrases Absent or artificially inserted
Room Acoustics Consistent background noise Inconsistent or sterile background
Formant Transitions Smooth spectral transitions Abrupt changes, especially in long vowels
Silence Analysis Micro-movements in "silence" Perfect digital silence

Python Spectral Analysis Script

"""
Audio Deepfake Detection - Spectral Analysis
Analyzes WAV files for synthetic audio indicators
"""
import numpy as np
import librosa
import librosa.display
import matplotlib.pyplot as plt
from scipy import signal

def analyze_audio_authenticity(audio_path: str) -> dict:
    """
    Perform forensic analysis on audio file.
    Returns dict of detection indicators.
    """
    # Load audio
    y, sr = librosa.load(audio_path, sr=None)
    
    results = {}
    
    # 1. High-frequency content analysis
    # Real speech has natural variation above 8kHz
    stft = np.abs(librosa.stft(y))
    freq_bins = librosa.fft_frequencies(sr=sr)
    high_freq_idx = freq_bins > 8000
    high_freq_energy = np.mean(stft[high_freq_idx, :])
    total_energy = np.mean(stft)
    hf_ratio = high_freq_energy / total_energy
    
    results['high_freq_ratio'] = hf_ratio
    results['hf_suspicious'] = hf_ratio < 0.01  # Very low = suspicious
    
    # 2. Silence analysis - synthetic audio has "perfect" silence
    silence_threshold = 0.01
    silent_frames = np.abs(y) < silence_threshold
    if np.any(silent_frames):
        silent_segments = y[silent_frames]
        silence_variance = np.var(silent_segments)
        results['silence_variance'] = silence_variance
        results['silence_suspicious'] = silence_variance < 1e-10
    
    # 3. Spectral flux - abrupt changes indicate splicing
    spectral_flux = librosa.onset.onset_strength(y=y, sr=sr)
    flux_std = np.std(spectral_flux)
    results['spectral_flux_std'] = flux_std
    results['flux_suspicious'] = flux_std > 2.0  # Abnormal variation
    
    # 4. Breath detection
    # Real speech has characteristic breath patterns
    rms = librosa.feature.rms(y=y)[0]
    breath_candidates = (rms < 0.05) & (rms > 0.001)
    results['breath_ratio'] = np.mean(breath_candidates)
    results['breath_suspicious'] = results['breath_ratio'] < 0.02
    
    # Overall assessment
    suspicion_count = sum([
        results.get('hf_suspicious', False),
        results.get('silence_suspicious', False),
        results.get('flux_suspicious', False),
        results.get('breath_suspicious', False)
    ])
    
    results['suspicion_score'] = suspicion_count / 4.0
    results['verdict'] = 'SYNTHETIC' if suspicion_count >= 2 else 'LIKELY_REAL'
    
    return results

# Usage
if __name__ == "__main__":
    results = analyze_audio_authenticity("suspect_audio.wav")
    print(f"Verdict: {results['verdict']}")
    print(f"Suspicion Score: {results['suspicion_score']:.2%}")

Facial Landmark Analysis with OpenCV

"""
Video Deepfake Detection - Facial Landmark Analysis
Detects temporal inconsistencies in facial features
"""
import cv2
import mediapipe as mp
import numpy as np

def analyze_video_authenticity(video_path: str, sample_rate: int = 5) -> dict:
    """
    Analyze video for deepfake indicators using facial landmarks.
    """
    mp_face_mesh = mp.solutions.face_mesh
    face_mesh = mp_face_mesh.FaceMesh(
        static_image_mode=False,
        max_num_faces=1,
        min_detection_confidence=0.5
    )
    
    cap = cv2.VideoCapture(video_path)
    fps = cap.get(cv2.CAP_PROP_FPS)
    
    landmark_history = []
    blink_frames = []
    frame_count = 0
    
    while cap.isOpened():
        ret, frame = cap.read()
        if not ret:
            break
        
        if frame_count % sample_rate != 0:
            frame_count += 1
            continue
        
        rgb_frame = cv2.cvtColor(frame, cv2.COLOR_BGR2RGB)
        results = face_mesh.process(rgb_frame)
        
        if results.multi_face_landmarks:
            landmarks = results.multi_face_landmarks[0]
            
            # Extract key landmark positions
            nose_tip = landmarks.landmark[4]
            left_eye = landmarks.landmark[159]
            right_eye = landmarks.landmark[386]
            
            positions = [
                nose_tip.x, nose_tip.y,
                left_eye.x, left_eye.y,
                right_eye.x, right_eye.y
            ]
            landmark_history.append(positions)
            
            # Blink detection (eye aspect ratio)
            ear = calculate_eye_aspect_ratio(landmarks)
            if ear < 0.2:  # Eye closed threshold
                blink_frames.append(frame_count)
        
        frame_count += 1
    
    cap.release()
    
    results = {}
    
    # Analyze landmark temporal consistency
    if len(landmark_history) > 10:
        landmarks_array = np.array(landmark_history)
        frame_diffs = np.diff(landmarks_array, axis=0)
        jitter = np.std(frame_diffs)
        
        results['landmark_jitter'] = jitter
        results['jitter_suspicious'] = jitter > 0.015  # Threshold from research
    
    # Analyze blink patterns
    if len(blink_frames) > 2:
        blink_intervals = np.diff(blink_frames)
        blink_regularity = np.std(blink_intervals)
        avg_blink_rate = len(blink_frames) / (frame_count / fps) * 60  # per minute
        
        results['blink_rate_per_min'] = avg_blink_rate
        results['blink_suspicious'] = (
            avg_blink_rate < 5 or  # Too few blinks
            avg_blink_rate > 40 or  # Too many
            blink_regularity < 5  # Too regular (mechanical)
        )
    
    # Verdict
    suspicion_count = sum([
        results.get('jitter_suspicious', False),
        results.get('blink_suspicious', False)
    ])
    results['verdict'] = 'SYNTHETIC' if suspicion_count >= 1 else 'LIKELY_REAL'
    
    return results

def calculate_eye_aspect_ratio(landmarks) -> float:
    """Calculate Eye Aspect Ratio for blink detection."""
    # Simplified EAR calculation
    left_eye_top = landmarks.landmark[159]
    left_eye_bottom = landmarks.landmark[145]
    vertical = abs(left_eye_top.y - left_eye_bottom.y)
    
    left_eye_inner = landmarks.landmark[133]
    left_eye_outer = landmarks.landmark[33]
    horizontal = abs(left_eye_inner.x - left_eye_outer.x)
    
    return vertical / (horizontal + 1e-6)

5. AI-Generated Image Detection

Common AI Image Artifacts

  • Asymmetric features: Earrings, glasses, eyes don't match
  • Hair anomalies: Blurry strands, unnatural patterns
  • Background distortion: Text, patterns become illegible
  • Finger/hand errors: Wrong number of fingers, fused digits
  • Teeth irregularities: Too uniform or bizarrely shaped

7. SOC Integration & Workflows

Deepfake Triage Workflow

SOC Analyst Checklist for Suspected Deepfake:
  1. Preserve original media file (chain of custody)
  2. Extract and document metadata (EXIF, container format)
  3. Run automated detection tools (Sensity, Video Authenticator)
  4. Perform manual spectral/visual analysis if automated results inconclusive
  5. Cross-reference claimed identity through out-of-band verification
  6. Document findings in incident report with confidence level
  7. Escalate to fraud team if synthetic media confirmed

8. Limitations of Current Detection

Detection Arms Race:
  • Detection accuracy degrades as generation models improve
  • Real-time deepfakes are harder to analyze (no file to examine)
  • Compression artifacts can mask both real and synthetic indicators
  • No single tool achieves 100% accuracy—layer multiple approaches
  • Legal admissibility of AI-based detection varies by jurisdiction
